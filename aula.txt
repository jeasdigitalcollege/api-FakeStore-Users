API REST

Padrão REST trabalha com os métodos/Verbos HTTP

Utiliza os métodos:
GET = Buscar Informações
POST = Salvar Novas Informações
PUT/PATH = Atualizar informações (PUT=documento completo) (Path= atualiza parte do documento)
DELETE = Deletar Informações.

Padrão de URL:

https://api.digital.com/alunos 

UUID = id utilizado nas aplicações modernas (Pesquisar Internet)

Passos para usar o simulador de API:

1. Instalar a extensão Thunder Client no VSCODE;
Thunder Client
Website - www.thunderclient.com
Documentation: docs.thunderclient.com
Support: github.com/rangav/thunder-client-support

2. Instalar o node,js
3. Instalar o servidor json-server (biblioteca JS):
	npm install -g json-server (-g= Geral)
4. Apresentar o arquivo que simula um servidor: 
	json-server --watch database.json
5. Mostrando todos os registros:
	GET http://localhost:3000/alunos
6. Mostrando o aluno com id=1
	GET http://localhost:3000/alunos/1
7. Adicionando o registro com id=7
	POST http://localhost:3000/alunos
8. Atualizando o registro com id=1
	PUT http://localhost:3000/alunos/1
9. Excluindo um aluno:
	DELETE http://localhost:3000/alunos/7
